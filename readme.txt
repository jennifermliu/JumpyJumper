Jumpy Jumper

For every level, you need to jump to the target block in specified number of jumps. Otherwise, you die.

Try to jump to the center of next block consecutively to get more points.

Left/Right Arrow Keys to control the direction bar

Space Key to control the jumping distance

Key Z to zoom in to visualize the position of the target  block

Key P to show the menu

Pay Attention to various types of blocks (See Next Page)
